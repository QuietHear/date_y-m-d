# 说明文档
> ### 使用步骤
>> 1. 下载插件
>> 2. 引入JQ
>> 3. 引入插件JS与CSS
>> 3. input上需要添加`class='date_picker' readonly`属性
>> 4. `<script>$('.date_picker').date_input();</script>`给输入框添加事件即可
> ### 兼容性
>> 兼容到IE10
